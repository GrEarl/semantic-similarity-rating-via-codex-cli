# 要求
./2510.08338v1.pdfの内容に基づき、GPT-5モデルを用いてSSR比較を行うWebアプリケーションを実装する

SSR比較ライブラリリファレンス実装のは https://github.com/pymc-labs/semantic-similarity-rating を用いる
GPT-5モデルへの操作は、Codex CLIをcodex execで発火して用いる

Codex Exec Example:
```bash
codex exec --model gpt-5 --json --sandbox read-only --skip-git-repo-check "<Question Here>"
```

# 仕様
バックエンドは FastAPIを用い、Codex execに対するジョブの管理、ペルソナ群の保存、結果レポートの生成を行う
フロントエンドはNext.jsを用いる
次のポイントが求められる
・Codex execは同期的なので、Webアプリとして破綻しないようFastAPIバックエンド側でジョブを管理する
・Web UIから、ペルソナや戦略をどのようにプロンプトに挿入するかを調整できる
・SSR比較に用いる語群テーブルの調整機能
・各戦略ごとの結果レポートペルソナ年代ごとの平均評価や各種統計的表・グラフで表示し、それを画像もしくはpdfでダウンロードできる機能
・ペルソナの管理機能(年代・性別等)
